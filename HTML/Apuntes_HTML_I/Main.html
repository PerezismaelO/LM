En Html hay muchisimas etiquetas, y ya las mayoria se aprenden de trabajar mucho con ellas. Sin embargo, siempre hay algunas etiquetas mas peculiares que se nos pueden olvidar, estas las dividiremos en:

Etiquetas de espacios en blanco (?)

Etiquetas de texto:

<em></em>
Sirve para dar enfasis al texto
<strong</strong>
Pone como fuerte el texto
<ins></ins>o <del></del>
ins es para marcar el texto nuevo añadido y del para indicar texto que ha sido borrado/eliminado
<blockquote></blockquote> y <cite></cite>
cite se le asigna al autor que dice una frase y blockquote a la frase que dice
<abbr></abbr>, <dfn></dfn> y <accronym></accronym>
abbr es para indicar abreviaturas, dfn para indicar una definicion, y accronym para indicar un acronimo
<code></code>
Sirve para indicar que es codigo

UD 7: Formularios

Los formularios se crean de la siguiente forma:

<form action="http://midominio.com/altas" method="post" enctype="multipart/form-data"></form>

Según lo que tenemos, vemos que form contiene un action donde está el receptor de los datos que le vamos a pasar

el method que puede ser get o post, get es el que tiene por defecto y el que usamos siempre, a no ser que el formulario contenga archivos o información privada como contraseñas, entonces cambiamos el metodo get por el metodo post.

enctype solo se pondría si en el formulario enviamos archivos
